# TradePro — Traders Website

A modern one-page website for **traders**, built with **React**, **Vite**, and **Bootstrap 5**.  
Features hero carousel, portfolio, services, testimonials, contact form, and responsive layout.

## Run locally

```bash
cd traders-website
npm install
npm run dev
```

Then open **http://localhost:5173** in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Stack

- **React 18** + **Vite**
- **Bootstrap 5** + **React Bootstrap**
- **Unsplash** images (trading, crypto, charts, analytics, support)
- Custom CSS (teal/gold theme, cards, hero)

## Sections

1. **Navbar** — Fixed, scroll-aware, links + Get Started
2. **Hero** — Full-height carousel with 3 slides
3. **Portfolio** — Stocks, Crypto, Forex cards with images
4. **Services** — P&L Analysis, Market Research, 24/7 Support
5. **Testimonials** — Carousel of trader quotes
6. **Contact** — Form + address, email, phone
7. **Footer** — Links, social, copyright

All images are loaded from Unsplash CDN; no local image assets required.
